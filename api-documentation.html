Documentation for the RESTful API

Text-to-AMR

Evaluates a sentence and returns a serialized AMR graph in PENMAN notation.

Method

GET or POST

URL

/api/text-to-amr?sentence=sentence

Parameters

Name Type Description
sentence String The sentence to evaluate.

Request Example

http://SPRING_URL/api/text-to-amr?sentence=This%20is%20an%20example%20sentence.

Response

{
    "penman": "(z0 / sentence
                    :mod (z1 / example)
                    :domain (z2 / this))"
}

AMR-to-text

Evaluates a serialized AMR graph in PENMAN notation and returns a sentence.

Method

GET or POST

URL

/api/amr-to-text?penman=penman

Parameters

Name Type Description
penman String The serialized AMR graph to evaluate.

Request Example

http://SPRING_URL/api/amr-to-text?penman=(z0%20%2F%20sentence%0A%20%20%20%20%3Amod%20(z1%20%2F%20example)%0A%20%20%20%20%3Adomain%20(z2%20%2F%20this))

Response

{
    "sentence": "This is an example sentence."
}
